Household Organic Waste Compost Subscription & Collection Tracking System

This project is a Java JDBCâ€“based application designed to manage household organic waste compost subscriptions and daily waste collection activities. It allows registering subscribers, tracking waste collection transactions, calculating collection fees, managing pending payments, and handling subscription status.

The system follows a layered architecture using Bean, DAO, Service, and Utility classes, with data stored in an Oracle database. It includes validations, custom exceptions, and transaction management to ensure data consistency and business rule enforcement.

Key Features:

Subscriber registration and management

Daily waste collection tracking

Automatic fee calculation based on collected weight

Pending amount and payment status handling

Subscription activation/deactivation with validations

Technologies Used:
Java, JDBC, Oracle Database
